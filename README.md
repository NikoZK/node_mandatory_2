# Node mandatory 2
Auth application for the mandatory 2 assignment in my Node.js course.


# To Run this application you need to do a few quick steps

*Run <code>npm install<code> in both the client and server directories.*

*Create a .env file and set SESSION_SECRET inside server directory*

*In the server directory run <code>node database/createDatabase.js<code> to create a table and seed the rows.*
